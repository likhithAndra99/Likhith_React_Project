# Employee Management Dashboard

A modern React.js web application to manage employees with login, CRUD, search, filters and image upload.

## Features
- Secure Login (Mock Authentication)
- Employee Dashboard with stats
- Add, Edit, Delete employees
- Profile image upload with preview
- Search & Filters (Gender, Active/Inactive)
- Print employee list
- Data persistence using Local Storage

## Tech Stack
- React.js
- Vite
- JavaScript
- CSS
- LocalStorage

## Steps to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open `http://localhost:5173`

## Design Decisions
- Context API used for global state
- LocalStorage used instead of backend
- Reusable form for add/edit
- Clean folder structure for scalability
